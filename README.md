# projetos-site_android
site desenvolvido durante o curso de HTML5 e CSS3 do Curso em Vídeo (módulo 2 de 5).
Tema: Android (sistema, curiosidades e notícias).
Foco em layout responsivo e semântica moderna.

Objetivos
Aprender prática de estrutura semântica (header, nav, main, footer)
Desenvolver navegação simples com barra de menu
Trabalhar tipografia com text-align: justify para parágrafos
Estilizar com variáveis CSS (:root) e organização de cores
Usar margin e padding para alinhamento visual (layout fluido)
Inserir imagens com picture + srcset para responsividade
Incluir vídeo responsivo (fit em diferentes telas)
Manter compatibilidade mobile e desktop
Funcionalidades implementadas
Página inicial, curiosidades, notícias e fale conosco
Menu de navegação com links internos
Texto justificado para melhor leitura
Imagens redimensionáveis conforme largura da tela
Vídeo embed adaptável a diferentes resoluções
Layout fluido com medidas relativas (%, max-width, etc.)
CSS em style.css e arquivo organizando fontes, cores e espaçamentos
Tecnologias
HTML5
CSS3
Elementos semânticos (header, nav, main, section, footer)
Responsividade básica (media queries, picture, video ajustável)
