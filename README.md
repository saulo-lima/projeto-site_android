#  Projeto Android

Site desenvolvido durante o curso de **HTML5 e CSS3** do Curso em Vídeo (módulo 2 de 5).

 **Tema:** Android (sistema, curiosidades e notícias)  
 **Foco:** Layout responsivo e semântica moderna  

---

## Objetivos

- Aprender na prática estrutura semântica (`header`, `nav`, `main`, `footer`)  
- Desenvolver navegação simples com barra de menu  
- Trabalhar tipografia com `text-align: justify`  
- Estilizar utilizando variáveis CSS (`:root`)  
- Organizar layout com `margin` e `padding`  
- Implementar imagens responsivas (`picture` + `srcset`)  
- Inserir vídeo responsivo (adaptável a diferentes telas)  
- Garantir compatibilidade entre mobile e desktop  

---

## ⚙️ Funcionalidades

- Página inicial, curiosidades, notícias e "Fale Conosco"  
- Menu de navegação com links internos  
- Texto justificado para melhor leitura  
- Imagens adaptáveis conforme a largura da tela  
- Vídeo incorporado responsivo  
- Layout fluido com medidas relativas (`%`, `max-width`, etc.)  
- Organização de estilos em arquivo `style.css`  

---

## 🛠️ Tecnologias

- HTML5  
- CSS3  
- Elementos semânticos (`header`, `nav`, `main`, `section`, `footer`)  
- Responsividade básica (media queries, `picture`, vídeo adaptável)  

---

##  Contexto

Projeto desenvolvido como parte dos estudos no curso **Curso em Vídeo**, com foco na construção de páginas modernas, organizadas e responsivas.
